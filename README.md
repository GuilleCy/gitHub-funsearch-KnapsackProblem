# gitHub-funsearch-KnapsackProblem
El Knapsack Problem (KP) es NP-Duro; diseñar heurísticas efectivas requiere experiencia y prueba-error.   FunSearch permite explorar automáticamente el espacio algorítmico mediante evolución de programas.   Este proyecto investiga si un marco híbrido FunSearch + IAG puede generar heurísticas competitivas sin intervención manual.
